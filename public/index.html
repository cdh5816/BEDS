BEDS

Building Earthquake Detection System

BEDS는 건물에 설치된 지진 감지 센서를 통해
실시간으로 진동 상태를 모니터링하고, 이상 징후를 빠르게 알려주는 웹 서비스입니다.

아래 지도에서 BEDS가 모니터링 중인 건물을 누구나 확인할 수 있습니다.

관리자 패널 - 건물 등록

위도/경도는 네이버/카카오 지도에서 좌표를 확인해 입력하거나, 나중에 자동 변환 기능을 붙일 수 있습니다.

관리자 로그인

기본 계정: admin@beds.local / bedsadmin123!
로그인 후 BEDS가 모니터링할 건물 정보를 등록할 수 있습니다.
